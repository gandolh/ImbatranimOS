## Context

The finish line (corpus/wiki/decisions.md): friend-install bar, 2GB RAM
floor with zram, build-from-source distribution, semantic v1.0. This brief
is the hardening + QA pass across everything 01-06 built. Depends on all
prior briefs.

## What to do

1. **Low-RAM validation**: boot + install + browse (3 Firefox tabs, one
   YouTube video) in a 2GB VM with zram active; record idle/loaded RAM in
   corpus/wiki/status.md. If 2GB genuinely fails, the floor decision gets
   formally revisited (decisions.md + log entry), not silently bumped.
2. **Friend-install QA on real hardware**: write USB, install on at least
   one physical machine (UEFI+SB on) — Wi-Fi, audio, suspend/resume,
   Discover install, update notifier, Welcome app. This is the bar
   itself; a VM pass does not count.
3. **Docs as product**: README a non-Linux friend can follow — clone,
   one build command, flash USB, boot menu key hints, install
   walkthrough. Screenshots included.
4. Full clean-clone rebuild (`--clean`) to prove reproducibility; tag
   v1.0 in git.
5. Sweep the corpus: status.md updated, resolved open questions cleared,
   log entries for the release; wiki pages reflect what actually shipped.
