You log in to a Windows-7-classic desktop — taskbar, start menu, tray, desktop
icons — rendered modern-flat in black & white with a crimson accent. Behind the
retro chrome is a real Alpine machine. Here's everything on it.

**23 windowed apps** ship in v1, plus a set of platform surfaces (search,
notifications, app management) woven into the shell itself. Every heavy engine
loads as a lazy chunk, so none of this weighs down the boot bundle.

## The platform surfaces

Not every capability is a window. Three of them live in the shell itself — which
is what makes the collection feel like *one* desktop rather than a folder of
apps.

<CardGrid>
  <Card title="Global search launcher" icon="magnifier">
    A command palette plus a taskbar Search button, backed by a jailed, bounded
    backend file search (`/api/files/search` — no symlink follow, result-capped,
    opt-in content grep). Find a file, hit Enter, it reveals in Files.
  </Card>
  <Card title="Notification center" icon="bell">
    A persist-backed store with a public `notify()` on `@imbatranim/core`,
    bottom-right auto-dismiss toasts (errors stay sticky), and a tray bell with an
    unread badge and history. Clock alarms and calendar reminders are its first
    callers.
  </Card>
  <Card title="Add-on manager" icon="puzzle">
    A Settings "Apps" section backed by a per-user disabled-set — one
    `enabledApps` filter across Start menu, palette, and desktop, with an
    `openApp` guard. Settings, Files, and Terminal are non-disableable on purpose.
  </Card>
  <Card title="File preview pane" icon="document">
    An explorer-style toggleable pane in Files: text, images, and A/V preview
    natively, with a metadata-card fallback. Width and toggle are remembered.
  </Card>
</CardGrid>

## What ties it together

Two conveniences make the apps feel like a system instead of silos:

- **Open-with, by extension.** Double-click a file in Files and the right app
  opens it — an extension→app map routes `.md` to the Markdown Editor, code
  extensions to Monaco, images to the Image Viewer, and so on. New documents are
  born in Files too (right-click → New → Spreadsheet / Document).
- **Explicit-save everywhere.** Every editor shares one Save discipline: Ctrl+S,
  a dirty `•` marker, and a close-guard warning — no autosave surprises, no
  "Save As" maze.

<Aside type="tip">
  Everything here runs as the unprivileged `imbatranim` user, over a single
  authenticated port. The apps stream **data**, never pixels — see
  [the ideas that empower it](/ideas/) for why that one choice shapes all of the
  above.
</Aside>

## Try it

```bash
docker run -p 8080:8080 -v imbatranim-home:/home/imbatranim imbatranimos
```

Open **http://localhost:8080**, pick a password (the first screen is a setup
wizard, never a login — there is no default password), and you're on the
desktop. Full run, backup, and HTTPS instructions are in the
[project README](https://github.com/gandolh/ImbatranimOS#readme).
