## Four ideas, no more

<CardGrid>
  <Card title="Real, not simulated" icon="laptop">
    Terminal = a real PTY. Files = the real FS. Monitor = real `/proc`. Faking the system is off-soul.
  </Card>
  <Card title="One container, one port" icon="seti:docker">
    Alpine + Node serve the desktop and API on one authenticated port. The server streams **data**, never pixels.
  </Card>
  <Card title="No sudo, ever" icon="approve-check">
    Runs as the `imbatranim` user. argon2id password, optional TOTP, internet-exposable behind a proxy.
  </Card>
  <Card title="Lightweight is identity" icon="rocket">
    A slim image and a snappy desktop. A new dependency needs a reason.
  </Card>
</CardGrid>
