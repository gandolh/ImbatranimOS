## Regenerate

```bash
npm run docs      # from repo root — turbo: sync corpus + generate reference + build
```

<Aside type="note" title="Kept honest">
  TypeDoc flags every undocumented `@imbatranim/core` export as a build warning (flip `treatWarningsAsErrors` in `typedoc.json` to hard-fail). Compodoc runs `--coverageTest 35`; ratchet it up as coverage climbs.
</Aside>

**Intent is rendered from the [corpus](/wiki/decisions/); code shape is generated from the source. Neither is ever typed twice.**
