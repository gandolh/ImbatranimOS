# Screenshot tool add-on (flameshot-style)

An add-on for something like flameshot — a screenshot tool for the
in-browser OS.

## Context

Flameshot's core loop: grab the screen, select a region, annotate
(arrows, text, highlights), then save/copy. Here the "screen" is the
web desktop itself.
